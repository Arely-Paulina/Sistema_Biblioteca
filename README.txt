# Sistema_Biblioteca
Proyecto para mantenimiento de software

IMPORTANTE SEGUIR LAS INSTRUCCIONES PARA DISFRUTAR DEL SOFTWARE
1. Instalar un softaware que contenga un servidor local con conexión a base de datos
(se recomienda XAMPP, con versión mayor a la 3.2.4).
2. Accede a la carpeta de instalación de XAMPP. Encontrarás una carpeta llamada "htdocs". Entra a ella y descomprime TODOS LOS ARCHIVOS dentro de ésta.
3. Activa los módulos correspondientes al servidor local y a la base de datos.
4. En el navegador de tu preferencia escribe la siguiente ruta: http://localhost/phpmyadmin/
5. Una vez que cargue se abrirá una ventana de administración de bases de datos. Busca el botón llamado "<---Importar". En la sección "Archivo a importar" hay un botón para elegir la base de datos a importar.
6. Busca dentro de las carpetas descomprimidas una carpeta llamada "db", entra a ella y selecciona el archivo "sistemabiblioteca.sql".
7. Más abajo en la ventana del navegador asegúrate de que el formato esté seleccionado en SQL. Por último presiona "Continuar".
8. Ahora, en el navegador de tu preferencia, escribe la siguiente ruta: http://localhost/Sistema_Biblioteca-main
9. Escribe en los campos de Usuario y Clave "admin". Presiona el botón "Ingresar"
10. El producto ha sido instalado y esta listo para ser usado :)
